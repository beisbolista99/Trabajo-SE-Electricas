# Template de proyecto

Copie esta carpeta mediante `scripts/crear_proyecto.py`. No use el template como proyecto real.

## Carpetas

- `00_entrada`: documentos originales por tipo.
- `01_documentos_clasificados`: documentos clasificados.
- `02_extracciones`: textos, tablas, imagenes, compactos e indices.
- `03_calculos`: calculos reproducibles.
- `04_revision_tecnica`: analisis por agente.
- `05_observaciones`: observaciones emitidas.
- `06_respuestas`: respuestas a comentarios.
- `07_entregables`: entregables finales.
