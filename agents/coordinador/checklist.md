# Checklist del coordinador

- [ ] Proyecto identificado.
- [ ] Subestacion identificada.
- [ ] Codigo documental registrado.
- [ ] Revision o edicion registrada.
- [ ] Fecha registrada.
- [ ] Disciplina registrada.
- [ ] Tipo de documento registrado.
- [ ] Estado documental registrado.
- [ ] Fuente registrada.
- [ ] Nivel de confianza asignado.
- [ ] Se verifico que no se mezclen proyectos.
- [ ] Se uso documento compacto cuando existe.
- [ ] Se registraron paginas o laminas consultadas.
