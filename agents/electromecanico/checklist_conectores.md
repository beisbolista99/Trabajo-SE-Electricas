# Checklist de conectores

- [ ] Diferenciar barra, llegada de linea y equipos.
- [ ] Revisar material.
- [ ] Evitar par galvanico.
- [ ] Revisar angulo.
- [ ] Revisar compatibilidad dimensional.
- [ ] Revisar apernado versus compresion.
- [ ] Verificar cantidades.
- [ ] Comparar planta, detalles y listado.
- [ ] Detectar duplicaciones.
- [ ] Detectar descripciones heredadas.
