# Formato de salida documental

```text
<codigo_documento>/
|-- original.pdf
|-- metadata.yaml
|-- texto_completo_por_pagina.md
|-- indice_tecnico.md
|-- documento_compacto.md
|-- tablas_extraidas.md
|-- referencias_cruzadas.md
|-- inconsistencias_detectadas.md
`-- reporte_compresion.yaml
```

Toda afirmacion debe incluir:

```text
[Fuente: pagina X, tabla Y]
```
