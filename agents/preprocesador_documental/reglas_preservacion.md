# Reglas de preservacion

Se puede condensar:

- Portadas.
- Encabezados repetidos.
- Pies de pagina.
- Logotipos.
- Espacios vacios.
- Historiales extensos.
- Texto normativo duplicado.

Nunca se debe omitir:

- Codigo documental.
- Revision.
- Fecha.
- Pagina de origen.
- Formulas.
- Variables.
- Unidades.
- Valores utilizados.
- Resultados.
- Tablas criticas.
- Equipos.
- Conductores.
- Diametros.
- Areas.
- Ductos.
- Camaras.
- Cajas de agrupamiento.
- Porcentajes.
- Referencias cruzadas.
- Inconsistencias.
